1 Spring介绍​

1.1 为什么要学​

  • Spring技术是JavaEE开发必备技能，企业开发技术选型命中率>==90%==

  • 专业角度

    • ==简化开发==，降低企业级开发的复杂性
    • ==框架整合==，高效整合其他技术，提高企业级应用开发与运行效率

1.2 学什么​

  • 简化开发

    • ==IOC(反转控制)==
    • ==AOP(面向切面编程)==
      • ==事务处理==

  • 框架整合

    • MyBatis
    • MyBatis-plus
    • Struts
    • Struts2
    • Hibernate
    • ……

1.3 怎么学​

  • 学习Spring框架设计思想
  • 学习基础操作，思考操作与思想间的联系
  • 学习案例，熟练应用操作的同时，体会思想
